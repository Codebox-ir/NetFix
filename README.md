# NetFix

A lightweight network repair tool for fixing Windows connectivity issues instantly.

## 🚀 Overview

NetFix is a fast and portable utility designed to repair common Windows networking problems.  
It resets core components such as DNS, Winsock, IP stack, proxy settings, and more — all from one clean interface.

## ✨ Features

- Flush & repair DNS cache  
- Reset Winsock  
- Reset full TCP/IP stack  
- Renew/Release IP address  
- Reset proxy settings  
- Restart network services  
- Full network repair sequence  
- View detailed network info  
- Lightweight & portable (no installation required)

## 💻 Supported Systems

- Windows 10 (64‑bit)  
- Windows 11  
- May also work on Windows 8 / 8.1 (not fully tested)

## 🧠 Why NetFix?

Many connectivity issues require running multiple commands manually.  
NetFix automates the entire repair process into a clean and fast workflow — ideal for troubleshooting DNS failures, proxy misconfigurations, registry corruption, and post-malware cleanup.

## 🛠 Tech / Approach

- CLI-based Windows repair tool  
- Color‑coded menu UI (purple theme)  
- Uses native Windows networking commands  
- Fully portable with zero dependencies  
- Designed for speed, clarity, and reliability
